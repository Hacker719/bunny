bunny

A simple shell script wrapper around multiple package managers

About

Bunny is a cross-OS, user friendly package manager for UNIX

Switch to another package manager has never been so easy! Bunny has cute alternatives to commands Bunny is free as in free software Only need to learn a few basic commands to use bunny anywhere

Supported Managers

Arch Linux (pacman) AUR (Arch Linux) Helpers trizen yay aurman Debian/*buntu (apt-get) OpenBSD (pkg) Gentoo (emerge) Void Linux (xbps) NixOS - nix supported on any system GuixSD - guix supported on any system

Quick Start

Currently bunny can only be installed manually via git.

$ git clone https://gitlab.com/bunny-team/bunny

Once the repository has been cloned, you need to use GNU make
(almost definetly already installed on your system) to install
bunny to your system (globally).

$ cd bunny # make install

And you're done installing!

To get started, run

$ bunny help

to get a list of commands.
